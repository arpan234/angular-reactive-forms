Hands-on Exercise

  1. Create a form using Form Builder
  2. Form will have two input fields: First name and Last Name
  3. Add the subgroup named Address which will have City and State inputs
  4. Bind the template form to the model
  5. Populate the form with initial values on component load
  6. Make all the form controls required and set minimum length to 2
  7. Display error for first and last name if not valid
  8. Create a button named Change City. When clicked, change the name of the city
  9. Use valueChanges event to detect if first name is set to Test, log an output in the console that Test names are not allowed
  10. Add submit button which will be disabled until the form is invalid
  11. Add Reset button which will reset the form
